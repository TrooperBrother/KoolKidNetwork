How to make a movie.

Getting something for the movie.

First thing first you need to get something to use in the movie, a recording of you and your friends doing something funny or gameplay footage of a school friendly video game.

Editing the footage on a Windows system.

If you are using Windows 7 or older you can use windows movie maker to add a title and edit the clip to your liking. If you are using windows 10 or above there is a built in video editor in windows 10. Just search for it in the windows search box.



Editing the footage on a Mac system.

Mac systems come with imovie which is a good editing software to use, you can download a video editor if you need.



Editing the footage on a chromebook.

You can use any video editor that you can get at the chrome store. Or a online editor.
