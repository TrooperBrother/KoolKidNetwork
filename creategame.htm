Want to make your own game?

There are a few tutorials on how to make wick editor games, these games can be made without any real knowing of code. Just search up Wick editor and go to the official site. There are tutorials there, that use interactive animation as a step stool to make point-and-click games.

But wait!

There are some guidelines that you need to follow when creating games. These guidelines are the following.

  • Make sure your game uses its assets, it may use images. You can go to remove it. bg to remove the background of images to use.
  • Make sure the content of your game is school friendly, I don't want this site taken down!

    • If you have made your game then please submit it on the games page.
