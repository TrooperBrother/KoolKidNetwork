Want to make your own game?

There is a few tutorials on how to make wick editor games, these games can be made without any real knowing of code. Just search up wick editor and go to the offical site. There is tutorials there, use the interactive animation as a step stool to making point and click games.

But wait!

There are some guide lines that you need to follow when creating games. These guide lines are the following.

  • Make sure your game uses its own assets, it may use images. You can go to remove.bg to remove the background of images to use.
  • Make sure the content of your game is school friendly, I don't want this site taken down!

    • If you have made your game then please submit it on the games page.
